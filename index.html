pluggable-telegram-bot

Pluggable Telegram Bot written in Node.

A work in progress framework for writing Telegram bot plugins in Node.js. The purpose of this is to allow users to combine multiple Telegram bot back-ends to run on one Telegram bot without them conflicting.

Usage

Copy config.template.json to config.json and add your Telegram bot token to it, then run:

node index.js
